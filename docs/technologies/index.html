Technologies

To accommodate for a growing collection of endangered web entertainment, Flashpoint comes with several open-source technologies:


Flashpoint Launcher

The Flashpoint Launcher (FPL) is a desktop application used for browsing, storing and launching other applications (in this case, web content). It is specifically developed for Flashpoint as a replacement for a modified LaunchBox distribution; however, it can be used for other purposes, as with the Kahvibreak Project.


Shockwave Projector Redirector

The Shockwave Projector Redirector (SPR) allows Shockwave movies to be played via a local proxy server. This is used alongside the Redirector to allow Shockwave games to run in Flashpoint Infinity.


You can find the source code for all of Flashpoint's projects on our Github page.
