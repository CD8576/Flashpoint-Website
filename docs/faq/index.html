Frequently Asked Questions

Q: What platforms do you support?
A: We support games and animations using Adobe Flash, Adobe Shockwave, HTML5, Java Applets, Unity Web Player, Microsoft's Silverlight, 3DVIA Player, 3D Groove GX, PopCap Plugin, ActiveX, Authorware Web Player and GoBit Games Plugin.

Q: What operating systems do you support?
A: Windows 7 and up, as thoroughly updated as possible.

Q: How many games/animations are there?
A: As of 6.3, we have over 28,000 games (and 2,000 animations). You can find the full list here.

Q: How can I request a game/animation?
A: You can make a request by checking the Master List first and then following the link listed there to the form.

Q: I want to contribute in some way (coding, adding games, testing, etc). How can I help?
A: Join our Discord server for more details. The majority of our communication happens there.

Q: Is Flashpoint open source?
A: Yes...ish. The source for our Launcher is here. We include source for the Redirector with every copy of Flashpoint (check the FPSoftware/Redirector folder). The sources for the software we include in Flashpoint are available from their respective websites (we don't modify the source to anything we include). The binaries for the web plugins themselves aren't open source, but if they were, we wouldn't be here to begin with.

Q: I can't launch any games.
A: Read this, then follow the troubleshooting steps in the readme. If you've done both of these and you're still having problems, ask for help in our Discord.

Q: I want a game removed from Flashpoint for one reason or another.
A: Alert us, either via our Discord or directly by email. We'll probably try to convince you to let us keep your game in for historical sake, but we aren't unreasonable.

Q: I have another question.
A: Consult our extended FAQ. If your question isn't answered there, join our Discord.
