Welcome to BlueMaxima's Flashpoint, the webgame preservation project.

Internet history is important, and content made on plugins like Adobe Flash and platforms like HTML5 are important not only to internet culture but games history as well. This project is dedicated to preserving as many games & animations as possible that run on web plugins and browsers, so that they aren't lost to time.

Flashpoint runs a combination of open source software and the original platforms that run this content, in an easy to use launcher. All the hard parts are taken care of thanks to our Redirector, which helps web content pretend it's on the internet. Find out more in the links to the left.

Download Flashpoint and play some internet history.
